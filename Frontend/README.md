# EcoLearn - Environmental Education Platform

EcoLearn is an interactive web platform designed to make environmental education engaging and practical for students. Built with React and Vite, it focuses on climate change, sustainability, and environmental protection through gamified learning experiences.

## Features

- **Interactive Environmental Quizzes**: Learn about climate change, biodiversity, renewable energy, and more
- **Real-World Eco Challenges**: Practical tasks like tree-planting and waste segregation
- **AI Eco Assistant**: Get instant answers about environmental issues
- **Eco-Points Dashboard**: Track your environmental impact and earn badges
- **Progress Tracking**: Monitor your learning journey in environmental topics

## Tech Stack

- **Frontend**: React 18 with Vite
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Animations**: Framer Motion
- **State Management**: React Context API

## Getting Started

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the development server:

   ```bash
   npm run dev
   ```

3. Open [http://localhost:5173](http://localhost:5173) in your browser
